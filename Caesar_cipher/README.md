# Шифр Цезаря

Программа на Python для шифрования текста с динамическим сдвигом. Каждое слово шифруется со сдвигом, равным его длине (количеству букв в слове).
Особенности алгоритма:
- Сдвиг для каждого слова вычисляется отдельно
- Регистр букв сохраняется
- Знаки препинания и пробелы остаются на месте
- Поддержка английского алфавитов

## Установка

```bash
git clone https://github.com/vera-melikhova/Caesar_cipher.git
cd Caesar_cipher
```

## Запуск

```bash
python3 Caesar_ciper.py
```

## Тестирование

```bash
python3 -m unittest test_Caesar_ciper.py -v
```

## Требования

- Python 3.6+
- unittest (встроенный модуль, дополнительной установки не требует)
